1) Краткое описание проекта (что мы собираем)

Небольшое полнофункциональное приложение «Feedback Manager»:

Фронтенд (Vue 3 + Vite)
Страница с формой обратной связи (минимум 5 разных полей), валидация на клиенте, отправка JSON на бэкенд. Страница списка — все отправленные обращения в виде таблицы, возможность поиска и просмотра JSON детали каждой записи. Приложение собирается в единые JS+CSS файлы (Vite/rollup).

Бэкенд (C#, ASP.NET Core)
REST API, которое принимает и сохраняет динамические формы (без жёстких моделей). Хранение — встроенная в проект база (не требует отдельного сервера): LiteDB или простой JSON-файловый стор (на выбор). Методы: POST /api/submissions, GET /api/submissions (с фильтром q), GET /api/submissions/{id}. CORS включен.

Part 2 (документ)
Текстовое описание архитектуры для больших вложений (≈1 страница): хранение файлов, chunked upload, CDN/облако, метаданные, API для скачивания и удаление/архивация.
